# Campus Pulse Student Satisfaction Dashboard

## Project Overview

Campus Pulse is an interactive student satisfaction analytics dashboard developed to analyze feedback related to campus facilities such as the Library, Cafeteria, and Sports Center.

The project helps visualize student satisfaction data using charts and metrics, enabling better understanding of student feedback and campus facility performance.



## Features

- Interactive dashboard
- Facility-based filtering
- Average satisfaction score analysis
- Total response tracking
- Rating percentage breakdown
- Facility performance comparison
- Dynamic charts using Chart.js



## Technologies Used

### Backend
- Python
- Flask
- pandas

### Frontend
- HTML
- CSS
- JavaScript
- Chart.js



## Dataset Fields

The dataset contains:

- student_id
- academic_year
- major
- facility_rated
- satisfaction_score
- timestamp



## Charts Used

- Bar Chart for Score Distribution
- Pie Chart for Rating Breakdown
- Facility Comparison Chart



## How to Run the Project

1. Download the repository

2. Open the project folder in VS Code

3. Install required libraries

```bash
pip install pandas flask flask-cors
```

4. Run backend

```bash
python app.py


5. Open `index.html` using Live Server

6. Select facility and click `Load Dashboard`
